# 🌍 LangLearn

A Flutter-based Language Learning Application developed as part of the **CodeAlpha Internship Program**.

LangLearn helps users learn new languages through interactive flashcards, quizzes, vocabulary exercises, pronunciation support, and progress tracking. The application provides an engaging and user-friendly learning experience for beginners who want to improve their language skills.

---

## 🎓 CodeAlpha Internship Project

This project was developed as part of the CodeAlpha Internship Program to demonstrate mobile application development using Flutter and Dart.

### Project Objectives
- Build a modern mobile application using Flutter
- Create an interactive language learning experience
- Implement quizzes and flashcard-based learning
- Track user progress locally
- Integrate text-to-speech for pronunciation support
- Design a clean and intuitive user interface

## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Android Studio or VS Code
- Android Emulator or Physical Device

### Installation

```bash
git clone https://github.com/your-username/CodeAlpha-LangLearn.git

cd CodeAlpha-LangLearn

flutter pub get

flutter run
```
